## 🚀 How To Use A Skill

2. Copy the **Prompt Block** at the top
3. Paste it into your AI (Perplexity, Claude, ChatGPT, HyperCode Agent)
4. Fill in the **[INPUT VARIABLES]**
5. Run it — get instant Hyper output ⚡
