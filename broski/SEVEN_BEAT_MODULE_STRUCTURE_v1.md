# HS-034 — 🎯 BEAT ARCHITECT — 7-Beat Module Teaching Structure
**Category:** broski
**Version:** 1.0

> *"Every module is a story. 7 beats = student goes from lost → capable → hungry for more."*

---

## 🎯 What It Does
The neurodivergent-first teaching structure used across all Hyper-Vibe-Coding-Course modules. 7 beats that take a student from zero context to a real win they can feel.

## 🌍 Why It Exists
Traditional tutorials lose ADHD learners in the first paragraph. This structure gives dopamine hits at every beat — keeping hyperfocus locked in.

## ⚙️ How To Use
1. Paste when writing or reviewing a course module
2. Fill in `[MODULE TOPIC]` and `[TECH USED]`
3. Write each beat in order — never skip STOP or WIN

---

## 📋 THE PROMPT

```
Write a course module using the 7-Beat ND-First structure for: [MODULE TOPIC]
Tech stack: [TECH USED]

BEAT 1 — STOP (Plain English Context)
  No code yet. 2-3 sentences max. What IS this thing in human words?
  Real-world reference (Netflix, Uber, Stripe, Discord).

BEAT 2 — WHY (Real-World Use Case)
  Why does this exist? What problem does it solve?
  Show the pain of NOT having it.

BEAT 3 — THE PATTERN (Visual Mental Model)
  One diagram, flowchart, or analogy.
  The "aha" moment before the code.

BEAT 4 — HOW (Step-by-Step with Time Estimates)
  Numbered steps. Each step has ⏱️ time estimate.
  Code snippets are small and focused — never walls.

BEAT 5 — BUILD IT (Hands-On Task)
  One concrete thing to build/run/test.
  Clear input → expected output.

BEAT 6 — WIN (Celebratable Moment)
  Explicit success state. What does "done" look like?
  Celebrate it: "Nice one BROski♾️! You just [achievement]."

BEAT 7 — NEXT (Warm Bridge)
  One sentence teasing the next module.
  Creates forward momentum, not an ending.
```

---

## ✅ Example
**Input:** MODULE TOPIC = "FastAPI Routes", TECH USED = "Python + FastAPI"
**Beat 6 WIN:** "Nice one BROski♾️! You just built your first AI-ready API endpoint. Stripe, Discord, and Netflix all started exactly here."
