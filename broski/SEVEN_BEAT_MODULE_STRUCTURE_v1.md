# HS-034 — 🎯 SEVEN BEAT MODULE — 7-Beat Module Teaching Structure

## What it Does
The neurodivergent-first teaching framework used in every Hyper-Vibe-Coding-Course module. 7 beats that take a learner from zero context to a real win — without overwhelm.

## When To Use
- Writing or rewriting any course module
- Creating any tutorial, guide, or walkthrough for ND learners
- Structuring AI explanations for ADHD/Dyslexic users

## THE STRUCTURE
```
7-BEAT MODULE TEACHING STRUCTURE:

Beat 1 — STOP (plain English context BEFORE any tech)
  → What world are we stepping into?
  → No code yet. No jargon. Just vibes.

Beat 2 — WHY (real-world use case)
  → Netflix does this. Uber does this. Stripe does this.
  → Why does this matter to YOU?

Beat 3 — HOW (step-by-step with ⏱️ time estimates)
  → Chunked steps. Each one completable in 2-5 mins.
  → No step longer than 1 screen.

Beat 4 — WIN (clear celebratable moment)
  → "You just did X. That's what engineers at Y do."
  → Explicit recognition. Not optional.

Beat 5 — CHECK (quick comprehension beat)
  → 1-3 question quiz OR a "spot the difference" moment
  → Low stakes. Confidence builder.

Beat 6 — NEXT (warm bridge to next module)
  → "Now you know X, next you'll learn Y — which is just X but for Z"
  → Never leave the learner hanging

Beat 7 — RESOURCES (optional deep dive)
  → 2-3 links max. No rabbit holes.
  → "If you want more, here's where to go"

Rules:
- STOP beat = mandatory. Never start with code.
- WIN beat = mandatory. Never skip celebration.
- Each beat = 1 short section. No walls of text.
- Analogies over jargon. Always.
```

## Example
> Module: "What is an API?"
> Beat 1 STOP: "Imagine a waiter at a restaurant..."
> Beat 2 WHY: "Every app you use — Spotify, Uber, Instagram — is just APIs talking to each other"
> Beat 3 HOW: Step 1 (2 min): Open browser → Step 2 (3 min): Call a real API...
> Beat 4 WIN: "You just called a real API. That's what every developer does."
